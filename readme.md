## Тестовое задание
Я подключил vue, поэтому обязательно установите все пакеты из списка команд по сборке проекта

## Настройка проекта
Для Laravel:
- `composer install`
- `copy('.env.example', '.env')`
- `php artisan key:generate`
- `php artisan migrate`
- `php artisan db:seed`
- `npm i && npm run dev`
- `phpunit`
